# PWA Icons

Para completar a implementação PWA, você precisa adicionar os seguintes ícones:

## Ícones necessários:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Screenshots necessários:
- screenshots/desktop-1.png (1280x720)
- screenshots/mobile-1.png (375x667)

## Como gerar:
1. Use sua foto de perfil ou logo
2. Ferramentas como https://realfavicongenerator.net/
3. Ou use a ferramenta PWA Builder da Microsoft

Por enquanto, o PWA funcionará sem os ícones, mas para produção é recomendado adicioná-los.
